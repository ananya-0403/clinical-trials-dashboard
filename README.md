# Clinical Trials Analysis Dashboard

## 📊 Overview
This project presents a Power BI dashboard analyzing global clinical trial data.

## 🔍 Key Insights
- Oncology is the leading research area
- Majority of trials are interventional
- Most studies are active or completed

## 📁 Files Included
- clinical-trials-dashboard.pbix
- cleaned_clinical_trials.csv
- queries.sql

## 🛠 Tools Used
- Power BI
- PostgreSQL (pgAdmin)
- Data Visualization
